# Event Manager Pro

A lightweight, mobile-first event management application built with pure HTML, CSS, and JavaScript. Perfect for organizing events, managing teams, and tracking payments - all without any external dependencies!

## Key Features

- 📝 **Create Events** - Add events with name, location, date, time, and payment details
- 👥 **Team Management** - Track incharge and volunteers with their contact numbers
- 🔔 **Smart Alerts** - Automatic notification for today's events
- 💾 **Persistent Storage** - All data saved locally in your browser
- 📱 **Responsive Design** - Works seamlessly on mobile, tablet, and desktop
- 🚀 **No Dependencies** - Pure vanilla JavaScript, no frameworks required
- 🌐 **Offline Ready** - Works without an internet connection

##  Perfect For

- Event organizers and planners
- Meetup and conference hosts
- Community managers
- Small business owners
- Workshop coordinators
- Anyone managing events!

##  Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Start managing your events!

No server setup, no installation, no registration required.
