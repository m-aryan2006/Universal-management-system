# Event Manager Pro

Event Manager Pro is a powerful, offline-first event management web application that helps you organize, track, and manage events efficiently. Built with vanilla JavaScript, HTML5, and CSS3, it stores all data locally in your browser using localStorage, ensuring your data is always accessible even without an internet connection.



🎯 Key Highlights
<br>
<br>
📱 Fully Responsive - Works seamlessly on mobile, tablet, and desktop

🌍 Multi-Language Support - 8 languages including English, Hindi, Spanish, French, German, Chinese, Arabic, and Japanese

📄 Professional Invoice Generation - Generate beautiful PDF invoices with company details

🗺️ Interactive Maps - Pick locations from map or enter addresses manually

🌙 Dark Mode - Toggle between light and dark themes

💾 Offline-First - All data stored locally in your browser

🚀 Installation
<br>
<br>
Option 1: Direct Download
bash
# Clone the repository
git clone https://github.com/yourusername/event-manager-pro.git

# Navigate to project directory
cd event-manager-pro

# Open index.html in your browser
open index.html
Option 2: Live Server (Recommended)
bash
# Install live-server globally
npm install -g live-server

# Run the application
live-server
Option 3: Any Web Server
Simply upload all files to any web server or open index.html directly in your browser.
